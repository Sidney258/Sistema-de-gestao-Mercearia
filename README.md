Sistema para gestao de uma Mercearia

Este foi o meu projecto final para a Disciplina de Programacao 3, onde foi pedido um sistema de gestao de Mercearia que realizasse as seguintes tarefas

- Gestao de Clientes
- Gestao de Produto e Armazens
- Gestao de Fornecedores
- Realizar Venda

O sistema apresenta muitas outras funcoes como: historico de vendas, produtos mais comprados e mais.
Sinta-se a vontade pra baixar o projecto, e se desejar, contribuir adicionando mais funcionalidades.
