# Sistema para gestao de uma Mercearia

Este foi o meu projecto final para a cadeira de Programacao 3. Desenvolver uma app desktop usando java swing e boas praticas de programacao orientada a objectos.
Uma aplicacao sem base de dados, onde os dados sao gravados em ficheiros.

## Funcionalidades
- Gestao de Clientes
- Gestao de Produto e Armazens
- Gestao de Fornecedores
- Realizar Venda

# Tecnologias Utilizadas
- java
  
## Requisitos
Java 12
